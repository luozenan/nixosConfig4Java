# nixosConfig4Java
这是一份nixos的配置文件，旨在提供一个开箱即用的java环境以及Java开发工具。
以下是目前使用的配置

桌面为gnome ,niri+dms, 默认为gnome，

jdk版本为1.8,默认路径/etc/jdk8

maven版本3.3.9,默认路径 /etc/maven

tomcat版本为tomcat9

文本编辑器：neovim，gnome-text-editor

文件管理：yazi，nautilus

终端：gnome-console

java开发工具为：idea ，vscode

数据库连接工具：datagrip

接口api：apifox

ssh客户端：electerm，finalshell

redis客户端：tiny rdm, another redis desktop manager

会议软件：wemeet（不好用，等原生的吧），jisti meet

office软件：wps

聊天：wechat，qq，wxwork(企业微信请自行安装，安装后续配置在bottles备注)

邮件客户端：thunderbird

window软件安装：bottles
