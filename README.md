# nixosConfig4Java
这是一份nixos的配置文件，旨在提供一个开箱即用的java环境以及Java开发工具。
以下是目前使用的配置

桌面为gnome

jdk版本为1.8

maven版本3.3.9

tomcat版本为tomcat9

java开发工具为：idea

数据库连接工具：datagrip

接口api：apifox

ssh客户端：electerm

redis客户端：tiny rdm

会议软件：wemeet，jisti meet

office软件：wps

聊天：wechat，wxwork(企业微信请自行安装，安装后续配置在bottles备注)

邮件客户端：thunderbird

window软件安装：bottles
